HUONG DAN CHAY GOC HOC TAP

1. Mo file chay_goc_hoc_tap.bat de chay ung dung.
2. Neu camera bao thieu thu vien, mo Command Prompt va chay:
   pip install opencv-python pillow
3. Khi Windows hoi quyen truy cap camera, hay bam Allow/Cho phep.
4. Du lieu se duoc luu trong file goc_hoc_tap.sqlite3 cung thu muc voi ung dung.
5. Anh chup tu camera se nam trong thu muc anh_camera.
